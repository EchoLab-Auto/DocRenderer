# ProDoc 文档系统

欢迎使用 **ProDoc**，这是一个基于 Vue 3 和 Neumorphism 设计风格的文档渲染与编辑框架。

## 特性

- 🎨 新拟态设计风格
- 📝 Markdown 实时渲染
- ✏️ 所见即所得编辑
- 🌓 亮色 / 暗色主题切换

## 快速开始

```bash
npm install
npm run build
npm run view ./docs
```

## 文档结构

```
docs/
├── README.md
├── guide/
│   ├── getting-started.md
│   └── configuration.md
└── reference/
    └── api.md
```
